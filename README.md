# Landing Page
The first project in the Professional Web Development Nanodegree from Udacity.

## Objective
Building a multi-section landing page, with a dynamically updating navigational menu based on the amount of content that is added to the page.

## Overview
- A dynamic web page consists of 4 sections, each section consists of some text.
- The page is provided with a nav bar in order to get to any section in the page once clicking on the specified section in a smooth way besides activating this section in the navbar with different color as well as changing the color of the section passed by while scrolling.
